# **Predictive maintenance workshop**

This repository contains the programming part of the predictive maintenance workshop offered by professors Daniel Straub and Nick Eleftheroglou. The provided jupyter notebooks and corresponding source codes are meant to support your understanding of predictive maintenance concepts via hands-on application of the material presented in the lectures.


## 🚀 **Getting Started**

We are using **🐍 Python** for the tutorials.  
To make this easier for you, we provide **Google Colab notebooks** ☁️, which allow you to run Python code directly in your browser — **no installation needed**.


> 💡 **Note:** To use Colab, you need a **Google account**.  


## 📘 **Notebooks**

####  **General Instructions**

1. Each tutorial has a link that opens the Jupyter Notebook directly in **Google Colab**.  
2. Once opened, go to **File → Save a copy in Drive** 💾 to save your personal version.  
3. Follow the step-by-step instructions inside the notebook.
